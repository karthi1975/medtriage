## API Testing Examples

### 1. Search for Patient by ID
```bash
curl -X POST http://localhost:8002/api/v1/patients/search \
  -H "Content-Type: application/json" \
  -d '{"query": "21003", "search_type": "id"}'
```

### 2. Get Patient History
```bash
curl http://localhost:8002/api/v1/patients/21003
```

### 3. Start MA Session
```bash
curl -X POST http://localhost:8002/api/v1/ma/session \
  -H "Content-Type: application/json" \
  -d '{
    "patient_fhir_id": "21003",
    "facility_id": 1,
    "chief_complaint": "chest pain",
    "ma_name": "Test MA",
    "specialty_id": 2
  }'
```

### 4. Run Intelligent Triage
```bash
curl -X POST http://localhost:8002/api/v1/ma/intelligent-triage \
  -H "Content-Type: application/json" \
  -d '{
    "session_id": "YOUR_SESSION_ID",
    "patient_fhir_id": "21003",
    "patient_name": "Miguel Garcia",
    "patient_age": 25,
    "patient_gender": "male",
    "symptoms": ["chest pain", "shortness of breath"],
    "facility_id": 1,
    "provider_name": "Dr. Smith",
    "specialty": "Cardiology"
  }'
```
